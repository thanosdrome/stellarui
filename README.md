# 🌟 Stellar UI

A lightweight, customizable React component library built with Tailwind CSS — designed for fast, accessible, and consistent UI development.

## 🚀 Features

- ✅ Ready-to-use React components (buttons, forms, modals, etc.)
- 🎨 Customizable via props and Tailwind utility classes
- ♿ Accessible by default (ARIA support, keyboard navigation)
- 📦 Lightweight with zero external UI dependencies
- 📘 Fully documented with live usage examples

## 📦 Installation

```bash
npm install stellar-ui
# or
yarn add stellar-ui
